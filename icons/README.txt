Required Icons for QuickClip:

1. icon16.png (16x16) - Used for the system tray icon
2. icon128.png (256x256 or larger) - Used for the application icon

Please replace these placeholder files with actual icons in PNG format with the following specifications:
- icon16.png: 16x16 pixels, PNG format
- icon128.png: At least 256x256 pixels, PNG format

The icons should be placed in this directory before building the application.
